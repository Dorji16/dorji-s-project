## More on laser cutting.
Laser cutting is the process of using a high-powered laser beam to cut or engrave materials such as:
1.Wood
2.Acrylic (plastic)
3.Cardboard
4.Glass (engraving only)

## Steps for laser cutting.
1. Choose a Design
Search for a design (eg: Pinterest,google)
Example: “bts plushies black and white”
Copy and paste the image into Inkscape
Use it as a reference to trace

2. Convert Image to Vector (Trace Bitmap)
Imported images have:
Low resolution
Blurry edges
Small unwanted spots
## How to fix this problem?
Go to Path, Trace Bitmap
This converts the image into a clean vector outline

3. Adjust the Settings
Use Brightness Cutoff
We have to adjust it until the outline becomes clear and unwanted details are removed from the picture.

4. Apply and Clean the Design
Click Apply
Use the cursor tool (top left) and drag the image.
A clean duplicate appears and the next thing you need to do is to delete the original image (with dots)

5. Create Cutting Boundary
Select the Rectangle tool and draw a box around your design which  defines the cutting area. 

6. Set Laser Cutting Parameters
Go to Engrave / Laser settings
Adjust includes:
Speed= depends on the materials. 
Power = depends on the materials.
Mode includes:
Engraving (surface)
Vector cutting (deep cutting)

7. Send to Machine (Print)
Click Print

## Reminders
1.Never touch the laser while printing.
2.Do not open the machine lid during cutting.
3.Exhauster On always.
4.Compressor On.
4.Stay near the machine while it is running
